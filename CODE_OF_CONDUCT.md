# Code of Conduct

This project follows a simple standard: be respectful, constructive, and generous with context.

Contributors are expected to:

- Use welcoming and inclusive language.
- Respect differing viewpoints and experiences.
- Give and receive feedback constructively.
- Focus disagreements on ideas and implementation details.

Unacceptable behavior includes harassment, personal attacks, discriminatory language, or sustained disruption of project work.

Project maintainers may remove comments, issues, pull requests, or contributors that violate this code of conduct.
